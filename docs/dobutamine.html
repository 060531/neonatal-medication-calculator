Dobutamine (250 mg / 5 ml) = 50 mg/ml

Order: Dobutamine mg

คำนวณจากสต็อก 250 mg/5 ml (50 mg/ml):

ใช้สูตร ml = mg ÷ 50 ⇒ ÷ 50 = ml

ปริมาตรรวมเป้าหมาย: ml

ส่วนผสมรอบแรก: ยา ml + ตัวทำละลาย ml

ผลลัพธ์หลังคูณ เท่า

ยา: ml × = ml

ปริมาตรรวม: ml × = ml

ดังนั้น ตัวทำละลาย: − = ml

☞ สต็อก: 250 mg/5 ml ⇒ 50 mg/ml

แนวคิด: ปริมาณยาที่ต้องการ (ml) = mg ÷ 50
