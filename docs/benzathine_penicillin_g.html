สูตรจากความแรงที่เลือก: mL = units / 150,000

แทนค่า: 0 / 150,000 = mL

Solvent

  • Sterile water 3.6 mL → ได้สารละลายรวม 4 mL (300,000 u/mL)
  • Sterile water 7.2 mL → ได้สารละลายรวม 8 mL (150,000 u/mL)

Therapeutic use (ตัวอย่าง)

  • Early syphilis: 2.4 MU IM single dose (แบ่งฉีดข้างละ 1.2 MU)
  • Late syphilis: 2.4 MU IM weekly × 3 doses

Route: IM เท่านั้น — Deep IM กล้ามเนื้อสะโพก อาจผสม 1% lidocaine 0.5–1 mL เพื่อลดปวดและสลับตำแหน่งฉีด
