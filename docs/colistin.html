Colistin : 150 mg / 2 ml

Order: Colistin 0.00 mg

สูตรจากสต็อก 150 mg / 2 ml → ml = (mg × 2) / 150

แทนค่า: 0.00 × 2 / 150 = ml

ผลลัพธ์หลังจากคูณ เท่า = 0.00 ml

☞ ขนาดยาแนะนำ: 2.5 – 5 mg/kg/day

Solvent

  • Sterile water 2 ml
  • จะได้สารละลายใส ไม่มีสี 2 ml
  • ความแรง 75 mg/ml

Solution

  • 0.9% Normal saline solution / NSS
  • 5% Dextrose in water

ความเข้มข้นสูงสุดในการบริหารยา

  • ≤ 3 mg/ml

ระยะเวลาในการบริหารยา

  • ≈ 30 mins

ความคงตัวหลังเจือจาง

  • ไม่เกิน 7 วัน ที่อุณหภูมิ 2–8°C

ข้อควรระวัง / การติดตาม

  • CBC
  • Renal function
  • Body temperature

Half-life: ~3 hrs
