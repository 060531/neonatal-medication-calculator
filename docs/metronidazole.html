Metronidazole : 500 mg / 100 ml (5 mg/ml)

Solvent

  • ยาอยู่ในรูป Solution (premix)
  • ความแรง: 5 mg/ml

Solution

  • สามารถให้โดยตรง ไม่ต้องเจือจาง (พิจารณาปริมาณรวมตามแผน)

ความเข้มข้นขณะให้ (สูงสุด)

  • ≤ 8 mg/ml

ระยะเวลาในการบริหารยา

  • ประมาณ 30 mins

ความคงตัวหลังเจือจาง

  • ~24 ชั่วโมง ที่อุณหภูมิห้อง (ขึ้นกับแนวปฏิบัติหน่วยงาน)

ข้อควรระวัง / การติดตาม เพื่อป้องกัน Type A ADR

  • Liver function
  • เฝ้าระวัง anaphylaxis ในการให้ครั้งแรก
